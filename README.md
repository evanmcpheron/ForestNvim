
# ForestVim

## Prerequisites

#### Install ripgrep
For further instructions please view the git repo [here](https://github.com/BurntSushi/ripgrep).
Arch
```
sudo pacman -S ripgrep
```

Linux
```
sudo apt-get install ripgrep
```

## KeyBindings

#### Update Plugins
```
:PackerSync
```

#### Go back to file explorer
```
<space>pv
```

#### Search Files
```
<space>pf
```
